#  Mitología Griega — Juego de Combate

Juego desarrollado en **Python + Tkinter** basado en combates entre dioses y criaturas mitológicas griegas.

## Descripción

El jugador selecciona un dios griego, lucha contra criaturas míticas, gana experiencia, sube de nivel y mejora estadísticas.

El juego incluye:

- Sistema de combate por turnos.
- Ataques normales y habilidades especiales.
- Sistema de maná.
- Inventario con objetos.
- Subida de nivel.
- Tabla Top 10 persistente usando JSON.
- Interfaz gráfica creada con Tkinter.

##  Instalación

### 1. Clonar repositorio

```bash
git clone URL_DEL_REPOSITORIO
```

### 2. Entrar al proyecto

```bash
cd proyecto
```

### 3. Instalar dependencias

```bash
pip install -r requirements.txt
```

### 4. Ejecutar programa

```bash
python main.py
```
